# Bitmoji-Faces-Gender-Classification

### 简介

- 这是来自HDU的深度学习课的一个大作业

- 采用了四种常见的分类模型完成，准确率99%以上

  - AlexNet
  - ResNet18
  - Inception

- 采用Pytorch
